What is new in 1.5.0?

In short, Networks can be now added/removed/excluded to the virtual machines. Improvements in the main window: Added "Add network" to the context menu of the selected virtual machine. Improvements in VM Inspector: Previously MAC address was shown. Instead, in this version, MAC address is shown in a new Networks table. The table shows mode, IP address and errors if any. NOTE: Displaying IP addresses of the virtual machines is a unique feature that is not seen in any other apps. Improvements on the Virtual Machine window: Each virtual machine now gets a new Network menu item to add/remove/edit network configuration. Backward compatibility: The datastructure of the virtual machine configuration is modified to accommodate multiple network configurations. Virtual Machine created with the previous app version can be opened by this version. But backward compatibility is not available.
